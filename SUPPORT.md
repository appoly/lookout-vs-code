# Support

Lookout is currently a preview. Bug reports and focused feature requests are
welcome in the [GitHub issue tracker](https://github.com/adamhulme/lookout-vs-code/issues).

Before opening an issue, check the README's requirements and known limitations,
then try the latest published version.

For a useful bug report, include:

- Lookout and VS Code versions;
- operating system and workspace type (local, WSL, SSH, or dev container);
- Codex CLI or Claude Code version, when relevant;
- a short reproduction and the expected/actual behavior;
- whether the workspace is trusted and whether the terminal opens in the editor
  or panel.

Please redact API keys, authentication material, custom command arguments,
working-directory names, repository content, and any `LOOKOUT_NOTIFY_TOKEN`
value. Lookout does not need terminal transcripts to diagnose lifecycle issues.

Do not report vulnerabilities in a public issue. Follow
[SECURITY.md](SECURITY.md) instead.
